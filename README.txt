Калькулятор
Базовая функциональность: "+", "-", "*", "/" и скобки
Дополнительная функциональность берется из .dll файлов, находящихся в папке plugins
Данная реализация включает дополнительные операции "sin()", "cos()", "pow()", "!"
Выражение вводится в командной строке
Калькулятор:
1) переводит введенное выражение в польскую запись
2) решает польскую запись
Можно было данные два шага совместить и выиграть в производительности
Но это губительно бы повлияло на читаемость кода и на его жесткость
Также из особенностей: string выделяет 15 байт даже под пустую строку, и поэтому хранить 1-2 символа в string не очень эффективно 
(при попытке уменьшить выделяемую память (void string::reserve()) ничего не происходит (может быть вопрос компилятора - не углублялся))
В пользовательских масштабах - не критично. Выбор string обусловлен более высокой степенью надежности.
Подключенные библиотеки отключаются в деструкторе калькулятора
Сборка проекта - через .sln файл (Visual Studio) (не забыть первым делом после клонирования репозитория "назначить Calculator в качестве запускаемого проекта")
В папке Calculator также находится CMakeList.txt для сборки CMake (надо папку plugins тогда скопировать в корневую над .exe директорию)
